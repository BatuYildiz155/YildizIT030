Mizan Finance Project

One project I have been working on is the Mizan Finance Project. The goal of this project is to create a professional online space where investors can buy and sell stocks, similar to platforms like Robinhood or Coinbase. We have also added an AI tool where users can ask questions about different stocks. Each stock includes a description to help users make informed decisions, especially if they want to know whether a stock aligns with their religious beliefs.

While building this website, I used HTML to organize the content and CSS to style the layout, navigation, images, and sections. I focused on making the website clean, easy to read, and consistent across all pages. This class and project helped me strengthen my coding skills and assist my friend, who is the main creator of the project, with improving the website.

This project is important because it shows my growth in web development and gives me a place to present my work in the future. I plan to continue improving the site by adding more links, options, and advanced AI settings.

Skills Used

  • HTML page structure
  • CSS styling
  • Navigation links
  • Website organization
  • Problem solving
  • Attention to detail

What I Learned

From this project, I learned how important it is to keep a website organized and consistent. I also learned how small design choices, such as spacing, colors, and navigation layout, can make a website easier to use. This project helped me become more confident with coding and gave me a stronger understanding of how websites are built.
